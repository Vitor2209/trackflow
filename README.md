# 🚚 TrackFlow

TrackFlow é uma plataforma full-stack para rastreamento e gestão de entregas, simulando um sistema real de logística.

O projeto foi desenvolvido com foco em colaboração, organização de código e arquitetura escalável, utilizando múltiplos serviços (backend, frontend e mobile).

---

## 📦 Estrutura do Projeto

```
trackflow/
├── backend/   # API (Node.js, Express, Prisma, PostgreSQL)
├── frontend/  # Interface web (React)
├── mobile/    # Aplicativo mobile (React Native)
├── docs/      # Documentação do projeto
```

---

## 🚀 Funcionalidades (Planejadas)

* 🔐 Cadastro e login de usuários
* 📦 Criação e gerenciamento de entregas
* 🚚 Status da entrega (Pendente, Em trânsito, Entregue)
* 📍 Rastreamento em tempo real
* 📊 Dashboard com resumo das entregas

---

## 🛠️ Tecnologias

## ⚙️ Como rodar o projeto

### 🔹 Backend

```bash
cd backend
npm install
npm run dev
```

### 🔹 Frontend

```bash
cd frontend
npm install
npm start
```

### 🔹 Mobile

```bash
cd mobile
npm install
npm start
```

---

## 📌 Regras do Projeto

* ❌ Não subir `node_modules`
* ❌ Não subir arquivos `.env`
* ✔️ Sempre usar `.gitignore`
* ✔️ Criar branches a partir da `main`
* ✔️ Nome de branch padrão: `feat/nome-da-feature`

---

## 🤝 Contribuição

1. Faça um fork do projeto
2. Crie uma branch (`git checkout -b feat/minha-feature`)
3. Commit suas mudanças
4. Abra um Pull Request
