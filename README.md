# 🚚 TrackFlow

**TrackFlow** é uma plataforma de rastreamento e gestão de entregas, desenvolvida como um projeto colaborativo para simular um sistema real de logística.

---

## 📱 Funcionalidades

* Cadastro e login de usuários
* Criação e gerenciamento de entregas
* Status de entrega (Pendente, Em trânsito, Entregue)
* Rastreamento em tempo real
* Dashboard com resumo das entregas

---

## 🧱 Tecnologias

### Backend

* Node.js
* Express
* Prisma
* PostgreSQL

### Frontend

* React
* Vite
* TailwindCSS

### Mobile

* React Native (Expo)
