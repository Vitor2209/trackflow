# 🤝 Contribuindo com o TrackFlow

Bem-vindo ao projeto!

## 📌 Regras

- Sempre escolha uma tarefa (issue) antes de começar
- Comente na issue avisando que vai trabalhar nela
- Crie uma branch (ex: feature/nome-da-task)
- Ao finalizar, abra um Pull Request
- Não faça commit direto na main

## 🚫 Inatividade

Se pegar uma tarefa e não atualizar, ela pode ser passada para outra pessoa.

## 🎯 Objetivo

Manter o projeto organizado e profissional.
